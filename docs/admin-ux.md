# Admin UX Standards

- Back Button Placement
  - All admin sub-pages: top-right header "← Back to Admin Dashboard"
  - All user dashboard sub-pages: top-right header "← Back to Dashboard"
  - Remove any bottom/back duplicates for consistency

- Admin Dashboard
  - Header with title + Back to User Dashboard (top-right)
  - Quick stats placed above sections; not sticky
  - Module cards in grid (2–3 columns responsive)

- Website Configuration
  - Super Admin only for writes
  - Text inputs mapped to ConfigKey/Value pairs

- Pages (CMS)
  - Downloads mirrors editor (Mega/Drive/Mediafire + custom)
  - Upload to public/downloads; auto-add mirror URL

- Events, Logs, Users, Payments
  - Use metallic cards, consistent headers and controls
  - Confirm destructive actions; show loading/disabled states
